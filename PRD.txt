# Product Requirements Document (PRD)

## Project Name

**Catify** – Chrome Extension that Replaces Every Image with Random Cats

---

# 1. Problem Statement

Users often browse websites with numerous images. This Chrome extension transforms the browsing experience by replacing every image on any webpage with a randomly fetched cat image.

The project demonstrates:

* Chrome Extension development
* Manifest V3 architecture
* DOM manipulation
* API integration
* Browser permissions
* Dynamic webpage handling

---

# 2. Goal

Develop a Chrome Extension that automatically replaces every image element (`<img>`) on a webpage with a random cat image fetched from a public API.

The extension must:

✅ Use Manifest V3

✅ Run on all websites

✅ Replace all existing images

✅ Handle images loaded dynamically after page load

✅ Be loadable via Chrome Developer Mode

---

# 3. Success Criteria

When the extension is enabled:

* Open Google
* Open GitHub
* Open Amazon
* Open YouTube

Every visible image becomes a cat.

No console errors.

No crashes.
